# node 版本控制

  1. 安装

    curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.1/install.sh | bash
    1

    验证

    command -v nvm
    1

    成功输出nvm

  2. 版本

    查看可以安装的版本

    nvm ls-remote
    1

    查看所有可以安装的LTS版本（长期支持版）

    nvm ls-remote --lts
    1

  3. 安装指定版本

    官方推荐的安装方式 nvm install <版本号>

    nvm install v5.5.0
    1

    但是推荐使用速度更快的方式：使用淘宝源安装

    NVM_NODEJS_ORG_MIRROR=https://npm.taobao.org/mirrors/node nvm install 6
    1

  4. 查看已经安装的 node

    nvm ls
    1

  5. 切换node 版本

    nvm use v6.10.2
    1

    验证node -v

  6. 设定默认node 版本

     nvm alias default v6.6.0
    1

    验证nvm current

  7. 卸载node 指定版本

    nvm uninstall v6.6.0
    1

# npm

  1. 镜像源

    npm get registry
    1

    设置成淘宝镜像

    npm config set registry http://registry.npm.taobao.org/
    1

    默认

    npm config set registry https://registry.npmjs.org/
    1

  2. 安装/卸载/更新

    npm install
    1
    npm uninstall
    1
    npm update
    1

  3. 过时

    npm outdated
    1
    npm outdated -g --depth=0
    1

  4. 查看已安装

    npm ls
    1

    查看全局包有哪些

    npm ls -g --depth=0
    1

  5. 参数

    --save-dev  //devDependencies （-D）开发环境
--save   //dependencies (可省略)
    1
    2
